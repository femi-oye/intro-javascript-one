Intro to Dev and Git Commands

Terminal Commands

  1. ls = Lists all files and directory in a given location
  2. cd = Change directory
  3. mkdir = Create folder in current directory
  4. touch = Creates a new file in current directory
  5. ls -a = Shows all files and hidden files

Git Commands

  1. git init = Initializes local git repo; only once
  2. git add . = Adds local changes to local repo. Behaves like a snapshot that prepares changes for commit
  3. git commit -m "message here" = Commit to Github
  4. git push origin master = Push to Github
  5. git remote add origin {http address for repo} = Add new repo to Github; only once
